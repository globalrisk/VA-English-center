# Blast game shooter images

Assets used by the Blast built-in game:

- `skibidi-head.png` — head only, transparent background. Rides the tip of the extending neck.
- `skibidi-toilet-base.png` — empty toilet with open bowl, transparent background. Fixed at the bottom of the arena.
- `skibidi-toilet.png` — original combined reference image (not loaded by the game).

If replacing: keep transparent backgrounds, roughly 256–512px, head facing up/left and toilet from a three-quarter front angle.
